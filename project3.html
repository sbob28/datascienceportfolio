Social Media Sentiment Analysis

This project involved analysing a large, compressed dataset of social media tweets, focusing on user demographics, geographic mentions, and sentiment analysis. By using command-line tools and programming languages like R, the project demonstrates efficient data processing and analysis techniques to extract insights from a massive dataset without requiring full decompression. Key tasks included determining the dataset’s structure and storage requirements, extracting unique user counts, identifying temporal data ranges, and conducting sentiment analysis for specific countries (USA and Canada) based on mentions in tweets. The sentiment results were then visualised to provide a clear understanding of public opinions, with potential applications in areas such as market research, trend analysis, and public sentiment studies. This approach showcases practical skills in handling large data files, data extraction, filtering, and visualisation.

This was the third assignment for a unit at Monash University

(Unit: Intro to Data Science)

Questions Explored

The project aimed to address these data processing and sentiment analysis tasks:

    A1. Inspecting the Data
    1. What is the size of the FIT1043_Dataset.gz file, and how can it be determined using shell commands?
    2. What delimiter is used to separate the columns in the file, and how was this deduced?
    3. How many lines are there in the dataset, and how can this be determined with a shell command?
    A2. Information from Data
    1. How many unique users are there in the dataset? Provide a shell command using "awk" and "uniq" to find this.
    2. What is the date range for the Twitter posts in this file?
    3. For each of the following tasks, provide a shell command and a brief explanation:
      • How many tweets mentioned the word “France” (case-insensitive)?
      • How many tweets mentioned variations of “France” (e.g., FRance, francE)?
      • Output the lines of tweets from the previous question into a file called myText.txt.
    A3. Data Aggregation
    1. Find the total number of negative, neutral, and positive tweets mentioning “USA” (case-insensitive) and do the same for “Canada”.
    2. Store the results of A3.1 in sentiment-USA.csv and sentiment-canada.csv files, formatted as CSV.
    3. Using R, read the sentiment-USA.csv and sentiment-canada.csv files, and create bar charts to visualise the sentiment for both countries.
    4. Discuss the differences between the two bar charts, analysing the sentiment distribution for the USA and Canada.
    A4. Small Challenge
    1. Extract timestamps of tweets mentioning “Australia” (case-insensitive) and save them into aus_time.txt using the shell.
    2. In R, read aus_time.txt and convert the timestamps to R datetime objects using strptime(). Provide an explanation of the format string used.
    3. Using the converted timestamps, calculate the number of tweets per day and create a histogram of daily tweet counts. Discuss the distribution of tweets in your histogram, noting any patterns.

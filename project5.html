Customised Chatbot

This project involves building a chatbot named Rhythm to assist customers of Mirchi Dance Crew (a dance crew that I am a part of). By utilizing Botpress as a primary development tool and Python for additional functionalities, the chatbot aims to streamline customer interactions, provide valuable recommendations, and enhance the overall user experience.

The goal is to help customers access information about upcoming workshops, performance packages for private shows, and provide tailored suggestions based on customer preferences. Rhythm also predicts package pricing based on customer requirements, ensuring transparency and convenience.

Technologies Used:
  • Platform: Botpress
  • Programming Language: Python
  • Libraries: Pandas, NumPy, Scikit-learn (for price prediction logic)

    • Results:
  • Best Feature: Accurate price prediction for performance packages based on customer inputs.
  • Customer Interaction Efficiency: Reduced manual effort by automating responses to frequently asked questions about workshops and package deals.

    • My Workflow

    Here is what my main workflow looked like.

    I wrote Python code for specific nodes to store variables, primarily to calculate the estimated price based on each customer's unique needs. Additionally, the code ensured the bot could repeat and confirm the customer's requirements to verify understanding.

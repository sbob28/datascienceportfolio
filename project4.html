Loan Approval Prediction

This project involves building a predictive model to determine the likelihood of loan application approvals based on applicant data. By employing machine learning techniques, the project aims to streamline the loan approval process, enhance decision-making accuracy, and reduce processing times for financial institutions.

The goal is to predict loan application outcomes (approved or not) based on key factors such as applicant demographics, financial data, credit history, and loan-specific attributes.

Technologies Used:
  • Programming Language: Python
  • Libraries: Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn
  • Tools: Jupyter Notebook

    • Results:

    Best Model: Gaussian Naive Bayes (GNB)

  • Accuracy: 82.93%
  • Minimal False Negatives (FN = 2), ensuring eligible borrowers are not overlooked.

    • Alternative Model: K-Neighbors Classifier

  • Accuracy: 78.86%
  • Balanced performance between false positives and false negatives.

    • Process / Approach

      Data Preprocessing:
      • Handled missing values for critical features (e.g., credit history and income).
      • Performed feature scaling using StandardScaler for numerical features.
      • Encoded categorical variables (e.g., gender, education) using LabelEncoder
      Exploratory Data Analysis:
      • Analysed trends in loan approval rates based on variables like gender, marital status, and credit history.
      • Visualised data distributions and relationships using histograms and bar plots.
      Model Selection:
      • Evaluated the following machine learning models:
          Random Forest
          Gaussian Naive Bayes (GNB)
          Decision Trees
          K-Nearest Neighbors (KNN)
      • Assessed accuracy, precision, and recall using confusion matrices.
      Key Priorities
      • (Performance Metrics):
          Minimized false negatives to avoid missing eligible borrowers.
          Balanced false positives to reduce financial risks.

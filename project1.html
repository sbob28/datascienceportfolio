GPA Influences Explorer

The GPA Influences Explorer project delves into a comprehensive dataset to analyse student performance metrics, focusing on the factors that impact GPA outcomes. This analysis encompasses various dimensions, including parental education and support levels, tutoring, study time, extracurricular involvement, and attendance. The project provides insights into how these factors correlate with academic performance, with a particular focus on identifying trends in student GPA distribution.

Through visualisations such as pie charts, histograms, box plots, and scatter plots, this project reveals key findings. For example, it demonstrates that while higher parental support and tutoring correlate positively with higher GPAs, absenteeism has a significant negative impact. Additionally, the analysis highlights the effects of extracurricular activities on academic success, showing that students engaged in multiple activities tend to achieve higher GPAs on average.

This project was developed using Python and data analysis libraries such as Pandas and Matplotlib. It serves as a valuable tool for understanding the multifaceted influences on student success, offering data-driven insights that could inform educational support strategies.

This was the first assignment for a unit at Monash University

(Unit: Intro to Data Science)
